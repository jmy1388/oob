# **App Name**: oob

## Core Features:

- 글 피드: 제목, 요약 및 작성자 정보와 함께 스크롤 가능한 글 피드를 표시합니다.
- 글 세부 정보 보기: 이미지 및 작성자 약력을 포함하여 글의 전체 내용을 표시합니다.
- 사용자 인증: 사용자가 계정을 만들고 로그인하여 독서 경험을 개인화하고 글을 저장할 수 있도록 합니다.
- 검색 기능: 사용자가 키워드, 작성자 또는 주제별로 글을 검색할 수 있도록 합니다.
- 개인화된 추천: 협업 필터링을 사용하여 읽기 기록을 기반으로 글을 추천합니다. 도구를 사용하여 사용자의 이전 활동 패턴에서 추천할 항목을 검색합니다.
- 글 저장: 로그인한 사용자는 자신의 개인 독서 목록에 글을 저장할 수 있습니다.
- 글 제출: 로그인한 사용자가 간단한 텍스트 편집기를 사용하여 자신의 글을 제출하고 게시할 수 있도록 합니다.

## Style Guidelines:

- 기본 색상: 차분하고 창의적인 느낌을 주는 부드러운 라벤더 (#D0BFFF)입니다.
- 배경 색상: 콘텐츠에 대한 중립적인 배경을 제공하는 밝은 회색 (#F0F0F0)입니다.
- 강조 색상: 하이라이트 및 클릭 유도를 위한 진한 보라색 (#6741D9)입니다.
- 제목 글꼴: 고급스러운 느낌의 세리프체인 'Playfair'입니다. 본문 글꼴: 제목을 보완하는 휴머니스트 산세리프체인 'PT Sans'입니다. 참고: 현재 Google 글꼴만 지원됩니다.
- 글을 보여주기 위해 깨끗한 카드 기반 레이아웃을 사용합니다. 가독성과 시각적 계층 구조를 우선시합니다.
- 탐색 및 글 작업(저장, 공유 등)에 간단하고 일관된 아이콘을 사용합니다.
- 새 콘텐츠 페이드 인 또는 상호 작용 강조와 같이 사용자 참여를 향상시키기 위해 미묘한 전환 및 애니메이션을 사용합니다.
